# SCP: SL Unofficial Classical Chinese Translation

A player-made Classical Chinese translation, compiled by the community.<br>

The current text is mostly vernacular, translated according to the source.<br>

> If you need to translate new content, add, delete, or modify based on the English source.<br>

## Installation

Use `git clone` to get the repository, then place the `lzh` folder into the game's `translation` directory.<br>

Run the command:<br>
```shell
git clone https://github.com/365KW/SCP_SL_LZH
```

## Contributing

Classical Chinese translation is difficult; please follow these guidelines for contributors:<br>

- [GitHub Contributing Guide](https://docs.github.com/en/get-started/exploring-projects-on-github/contributing-to-a-project)
- [Classical Chinese Wikipedia Style Guide](https://zh-classical.wikipedia.org/wiki/%E7%B6%AD%E5%9F%BA%E5%A4%A7%E5%85%B8:%E7%AB%A0%E6%B3%95)

Do not submit vernacular text, and do not submit machine-generated translations.<br>

> If you wish to participate, fork this repository.<br>
> Create a new translation branch, submit well-written Classical Chinese to the main branch, and open a pull request.<br>

See the [style guide](https://github.com/365KW/SCP_UNOFFICIAL_LZH/tree/main/%E8%AD%AF%E5%BA%AB%E4%BB%8B%E9%9D%A2%E6%96%87%E6%9B%B8) here — you must read it.<br>
